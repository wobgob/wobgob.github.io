AzerothCore for Docker DesktopDocker is a platform for helping people build, share and run applications. We assume that you are familiar with playing on a private server and have a 3.3.5a client. We'll be relying on Docker Desktop and in-game commands to run our server so minimal technical expertise is required.

The Docker imageA Docker image is like the executable you download off the internet. A Docker container is like the running process you get after double-clicking the executable. provided by Wobblin' Goblin has cross faction, bots and scaling to allow for a good single player or small group experience. Think of our Docker image as a repack that is easily updated.

Installation

You'll need Docker Desktop to get started. Download Docker Desktop and install it.Windows users should also install Windows Subsystem for Linux from the Microsoft Store and use it as the Docker Desktop backend. Once you have Docker Desktop running installing the server is done in three easy steps.

  1. Run Image

    In Docker Desktop go to the search bar (Ctrl+K) and search for wobgob/azerothcore under the Images tab. Once you've found the image you'll want to select it and then hit Run. This will will bring up a prompt. Do not hit Run again.

  2. Enter Settings

    In the prompt open Optional settings and make sure your settings match the below image and then hit Run.

    You'll be taken to the server's logs where you'll need to wait for the server to initialize before you can login. First-time setup can take a few minutes.

  3. Login

    Leave Docker Desktop and set the realm list of your 3.3.5a client to 127.0.0.1. Open your 3.3.5a client and then use the username acore and password acore to log into the administrator account.

Setup

Once you're logged into the administrator account create a game master (GM) character with a name of your choice. This will be the character you'll manage the server with.

It's recommended that you use a separate player account for playing by yourself or with others. To create another account enter the following commandSee AzerothCore's website for a list of GM commands. into chat

.account create $account $password

Where you should replace $account with your account name and $password with your account's password.

You can now log into your player account and begin playing!

FAQ

How do I get help? Join the community to get help with Docker Desktop or running your server.

How do I use the bots? Refer to the NPCBots README to learn how to use the bots.

How do I play with others? Unless you port forward or use a tunneling service you'll only be able to play with others in your local area network. You'll also need to update the address field in the acore_auth.realmlist database table with your external IP address.

How do I change the configuration? You'll need to download the code from wobgob/azerothcore. You can then edit the configuration in etc. When running your container create another volume and set the host path to the etc directory and the container path should be /azerothcore/etc.

How do I backup my data? You'll need to use a database tool to dump your database's contents. To connect to the database the server host should be 127.0.0.1, the port is 3306, the username is acore and the password is acore.

How do I contribute? The best way to contribute is to join the community to help others and to share and discuss your creations.
