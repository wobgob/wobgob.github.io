AzerothCore

Wobblin' Goblin's AzerothCore for Docker Desktop

AzerothCore for Docker Desktop is a great way to start playing a 3.3.5a emulator by yourself or with your friends and family. The only requirement is Docker Desktop which you'll need to have installed to follow this tutorial.

The binary comes with opinionated defaults to allow for a good single player or small group experience. However, the binary can be compiled from source to create a completely custom experience.
