AzerothCore for Docker DesktopDocker is a platform for helping people build, share and run applications. Docker Desktop is a graphical application for building and sharing applications and services on the Docker platform. is a great way to start an AzerothCore server for yourself or for your family and friends.

We assume that you are familiar with playing on a private server and have a 3.3.5a client. We'll be relying on Docker Desktop and in-game commands to run our server so minimal technical expertise is required.

The Docker imageA Docker image is a file used to execute code in a Docker container. A Docker container is a runnable instance of an image. provided by Wobblin' Goblin has bots, scaling and individual progression to allow for a good single player or small group experience. Think of our Docker image as a repack that is easily updated and that you can compile yourself to create your own experience.

Installation

You'll need Docker Desktop to get started. Download Docker Desktop and install it. Once you have Docker Desktop installing the server is done in three easy steps.

  1. Run Image

    In Docker Desktop go to the search bar in the top right (Ctrl+K) and search for wobgob/azerothcore. Once you've found the image you'll want to hit Run which will bring up a prompt. Do not hit Run again yet.

  2. Enter Settings

    In the prompt open Optional settings and make sure your settings match the below image then hit Run.

  3. Login

    You'll be taken to the server's logs where you'll need to wait for the server to initialize before you can login. First-time setup can take a few minutes.

    Set the realm list of your 3.3.5a client to 127.0.0.1 then use the username acore and password acore to log into the console account.

Setup

Once you're logged into the console account create a game master (GM) character with a name of your choice. This will be the character you'll manage the server with.

It's recommended that you use a separate player account for playing by yourself or with others. To create another account enter the following commandSee AzerothCore's website for a list of GM commands. into chat

.account create $account $password

Where you should replace $account with your account name and $password with your account's password.

You can now log into your player account and begin playing!

FAQ

How do I use the bots? Refer to the NPCBots README to learn how to use the bots.

How do I play with others? Unless you port forward or use a tunneling service you'll only be able to play with others in your local area network.

How do I backup my data? To backup your data you will need to expose port 3306 and use a database tool to dump its contents. To connect to the database the server host should be 127.0.0.1, the port is 3306, the username is acore and the password is acore.

How do I contribute? The best way to contribute is to join the community to help others and to share and discuss your creations.

Appendix: Compiling

Compiling yourself is a great way to create your own custom AzerothCore experience. To start compiling you'll need to complete the following steps.

  1. Join Github
  2. Download and install GitHub DesktopLinux users can find releases at shiftkey/desktop and Visual Studio Code
  3. Open GitHub Desktop and in the File menu click Clone Repository...
  4. Click URL and enter wobgob/azerothcore. Click Choose... and navigate to a local path where you want to clone the repository then click Clone.
  5. Click For my own purposes then click Continue
  6. After cloning, in the Repository menu click Open in Visual Studio Code
  7. In Visual Studio Code you can edit the configuration files in the etc folder
  8. After you have saved your edits, in the Terminal menu click New Terminal
  9. In the Terminal enter docker build -t azerothcore . to compile your image
  10. Compilation can take a while but once it's finished you can find your new image called azerothcore in Docker Desktop to run

This is just a bare minimum example of changing configuration files. You can also add modules from the AzerothCore Catalogue to create your own custom AzerothCore experience. For further assistance with creating your own custom AzerothCore experience join the community.
